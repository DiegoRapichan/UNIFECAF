## 🧠 Conteúdo

### 📝 Descrição Narrativa

Sequenciamento lógico passo a passo de tarefas do cotidiano:

1. Fazer um café coado
2. Configurar o despertador no celular
3. Ligar um computador
4. Carregar o celular
5. Assistir a um filme em casa
6. Regar uma planta
7. Fazer uma pipoca
8. Participar de uma reunião online
9. Criar uma lista com as tarefas da semana
10. Assistir a um vídeo do Diego no canal da Rocketseat 💜

---

### 🔷 Fluxogramas

Representação visual com formas padronizadas:

1. **Calcular o dobro de um número** — fluxo sequencial simples
2. **Verificar se número é menor que 10** — estrutura de decisão (se/senão)
3. **Contar até 5** — estrutura de repetição com loop e contador

---

### 💻 Pseudocódigo

Exercícios com foco em raciocínio lógico estruturado:

```
01. Imprimir um número em tela
02. Somar dois números
03. Contar de 1 até 10 usando loop
04. Verificar se o número é par ou ímpar
05. Verificar se número é positivo, negativo ou zero
06. Calcular média de 3 notas
07. Verificar se aluno foi aprovado (média ≥ 5)
08. Maior entre dois números
09. Mostrar os múltiplos de 3 até 30
10. Contagem regressiva de 10 até 1
```

## 🧩 Conceitos Abordados

- **Sequência** — execução linear de instruções

- **Repetição** — laços de iteração (`Enquanto`)
- **Variáveis** — armazenamento e manipulação de dados
- **Entrada/Saída** — leitura e exibição de informações
