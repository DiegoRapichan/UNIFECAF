# 🧠 Lógica Proposicional na Prática

Este repositório contém exercícios práticos de lógica proposicional aplicados ao contexto de programação.

O objetivo foi reforçar o raciocínio lógico por meio da construção de expressões e tabelas verdade — base essencial para estruturas condicionais em qualquer linguagem.

---

## 📌 Conteúdos abordados

- Conectivos lógicos:
  - AND (`∧`)
  - OR (`∨`)
  - Implicação (`→`)
  - Bicondicional (`↔`)
- Construção de expressões lógicas
- Tabelas verdade
- Interpretação de problemas em linguagem natural

---

## 🧠 Exemplos trabalhados

```text
p ∧ q
p ∨ q
p → q
p → (q ∧ r)
p → (q ∨ r)
p ↔ q

💻 Aplicação em programação

Esses conceitos são diretamente utilizados em código:

if (estudou && fezExercicios) { ... }

if (irCinema || ficarEmCasa) { ... }
```
